# Safety Note

This project is intended for defensive public-health preparedness research and safe evaluation only.

- It uses synthetic or clearly public non-sensitive data.
- It does not develop biological, laboratory, or offensive capabilities.
- It does not provide instructions, models, or workflows that could facilitate harm.
- It does not support medical decision-making, operational surveillance, or real-world deployment.
- Future work should remain within responsible research, privacy, and governance boundaries.
- Expert feedback and careful review are important for any expansion beyond this prototype.

The repository is deliberately narrow because responsible scoping is part of the work, not an afterthought.
